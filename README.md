# 🏥 UHAP – UITM Health Appointment System

## 📌 Overview
UHAP is a web-based health appointment system developed as part of a System Development course. It allows users to register, log in, and book medical appointments efficiently.

---

## 🚀 Features
- User registration & login system
- Appointment booking system
- Dashboard for users
- Admin management
- Database integration with MySQL

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Local Server: XAMPP
